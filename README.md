# Karafuru Photobooth

> A photobooth desktop application for Windows.

---

## Download

Head over to the [**Releases**](../../releases) page to download the latest version.

---

## Installation

1. Download the `.exe` file from the [Releases](../../releases/latest) page
2. Run the installer and follow the on-screen instructions
3. Launch **Karafuru Photobooth** from your desktop or Start Menu

> **Note:** If Windows shows a SmartScreen warning, click **More info → Run anyway**. This is expected for unsigned apps.

---

## Requirements

- Windows 10 or later (64-bit)
